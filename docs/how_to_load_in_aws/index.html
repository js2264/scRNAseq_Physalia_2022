How to log in AWS

Connect to pre-configured AWS machine

KEY="path/to/your/key.pem"
chmod 600 ${KEY}
#
USER= #e.g. user=user2
IP= # e.g. IP=255.255.255.255
#
ssh -i ${KEY} ${USER}@${IP}

E.g., for user1:

KEY="c1.pem"
chmod 600 ${KEY}
USER=user1
IP=54.188.62.211
ssh -i ${KEY} ${USER}@${IP}
